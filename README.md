# DevOps Immersion Workshop

## Azure DevOps, GitHub, and GitHub CoPilot

*Times are approximate and might be fluid. 
*This is an Intermediate to advanced (level 200 - 300) workshop with a lot of technical details. 
*Modules might be presented separately
